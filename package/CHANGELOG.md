# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-11-29

### Changed
- Renamed `types.ts` to `index.ts` for better package structure
- Updated package.json to export from `index.ts` instead of `types.ts`

## [1.0.0] - 2025-11-29

### Added
- Initial release
- Type definitions for Resend email webhooks
- `Email` interface
- `EmailAddress` interface
- `EmailAttachment` interface
- `EmailHeaders` interface
- `ResendWebhookPayload` interface
